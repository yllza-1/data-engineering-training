# Day 4 - Python and SQL Logic Review

## Goal of the Practice

The goal of this practice was to compare Python and SQL logic while working with order data. I practiced filtering, sorting, calculating values, and analyzing data using both Python and SQL.

The main focus was understanding how the same data operations can be done in different ways:

- Python uses loops, conditions, and dictionaries.
- SQL uses queries, filters, sorting, and aggregation functions.

## Files Included

- `python_orders.py` - Contains Python tasks for processing order data.
- `sql_queries.sql` - Contains SQL queries for filtering, sorting, and analyzing orders.
- `comparison.md` - Contains the comparison between Python and SQL logic.
- `daily_report.txt` - Contains the daily practice reflection.

## How to Run python_orders.py

1. Make sure Python is installed.
2. Open the project folder in the terminal.
3. Run the command:

```bash
python3 python_orders.py
```
